UserPassForce

Tipo: Ataque de Fuerza Bruta

Requisitos: Widnows 8.X, Windows 10-X

Descripción

Con este programa podréis poner a prueba la seguridad de las contraseñas de cualquier usuario en Windows. La función de éste es encontrar la contraseña del usuario seleccionado dentro de un archivo de texto (diccionario) Los diccionarios se pueden crear tanto manualmente como con programas externos o descargar ya creados.

Instalación

Al ser un conjunto de script nativos y compilados no se necesita instalar ningún programa..

Ejecución

Lo ejecutamos y nos saldra esta ventana.


Primero se deben completar los parámetros necesarios

En mi caso rellenare con mis datos:

User:eloy


Diccionario: Diccionario.txt

Ataque


Ejecutamos el ataque y esperamos.

Si tenemos sospechas de cómo puede ser la contraseña del usuario víctima con un poco de suerte y el diccionario correcto el programa encontrará la contraseña.

Resultados

En mi caso tenia el diccionario con la contraseña ya preparada para esta demostración, como podéis ver la contraseña ha sido hallada y seguidamente guardada en un archivo de texto dentro del directorio “datos\usuario”.


Observaciones

Este programa está creado mediante una idea creativa pero poco productiva si hablamos de tiempo, por lo que no se recomienda usar diccionarios muy amplios, estamos hablando de aproximadamente 1 - 3 contraseñas probadas por segundo.
