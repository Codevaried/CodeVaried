CAPT2

Tipo: Virus Spyware

Requisitos: Windows 10-X

Descripción

Este malware captura imágenes de la pantalla del usuario víctima y las envía a una dirección de Gmail en forma de paquetes o vídeos

Preinstalación

Para que este programa funcione correctamente y los archivos puedan llegar correctamente al Gmail destino, debemos permitir el acceso de aplicaciones poco seguras aquí enlace, después de permitir esto en la cuenta de Gmail destino ya podemos usar este programa correctamente.


Instalación

Una vez descargado el archivo comprimido, extraemos la carpeta "INSTA_CAPT2" preferentemente al escritorio del usuario víctima (equipo donde se echarán las capturas), seguidamente ejecutamos "CAPT2.exe" que se sitúa dentro de la carpeta extraída.


Configuraremos las opciones avanzadas según las necesidades y añadimos el Gmail destino.

Resultados

Como se puede apreciar en esta imagen los paquetes llegan correctamente según la confirmación definida en la instalación a la cuenta de Gmail destino


Observaciones

A la hora de añadir el Gmail destino se recomienda que no sea el personal por motivos de seguridad.

No se recomienda instalar este malware en varios ordenadores a la vez la cuenta de Gmail destino posiblemente sera bloqueada temporalmente por la abundante anomalía de conexiones en un corto periodo de tiempo.

Recuerda, se requiere que el usuario víctima tenga conexión a internet para que los paquetes puedan llegar correctamente al gmail destino.
